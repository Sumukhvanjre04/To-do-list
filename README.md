A simple yet powerful Python-based task manager designed to help students organize their daily tasks, track progress, and analyze productivity visually.

Technologies Used
-Python
-Pandas
-Matplotlib
-Tabulate
-JSON / File Handling
-Basic Data Structures

How to Run this Project:

1.Install Python
Make sure Python 3 is installed on your system.

2.Install required libraries:
pip install -r requirements.txt

3.Run a project:
python to_do_list.py
